# Expense Tracker
================

A simple expense tracking application built with React, using a context API for state management.

## Features

* Add income to your wallet
* Add expenses with categories (Food, Transport, Entertainment, Bills, etc.)
* View top expenses
* Delete expenses
* Edit expenses

## Components

* `Header`: Displays the application title, wallet balance, and buttons to add income and expenses.
* `Transaction`: Displays a list of expenses, with options to edit and delete each expense.
* `TopExpenses`: Displays a bar chart of top expenses.
* `AddIncomeModal`: A modal to add income to your wallet.
* `AddExpenseModal`: A modal to add expenses.

## Context API

The application uses a context API to manage state. The `ExpenseContext` is defined in `src/context/ExpenseContext.jsx` and provides functions to add income, add expenses, and delete expenses.

## Dependencies

* React
* React DOM
* React Icons
* Recharts
* Notistack

## Scripts

* `dev`: Starts the development server using Vite.
* `build`: Builds the application for production using Vite.
* `lint`: Runs ESLint on the codebase.
* `preview`: Starts a preview server using Vite.

## Styles

The application uses CSS modules for styling. The styles are defined in separate files for each component.

The application uses React Icons for icons.
